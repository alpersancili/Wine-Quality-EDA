# Wine-Quality-EDA
This repository contains an Exploratory Data Analysis of the Wine Quality dataset. The analysis helps to understand relationships between different physicochemical properties of wine and its quality. 

**Features in the dataset include:**
- `fixed acidity`
- `volatile acidity`
- `citric acid`
- `residual sugar`
- `chlorides`
- `free sulfur dioxide`
- `total sulfur dioxide`
- `density`
- `pH`
- `sulphates`
- `alcohol`
- `quality` (target variable)

## EDA Steps
1. **Data Loading and Inspection**
   - Checked dataset structure, types, and summary statistics  

2. **Data Cleaning**
   - Verified missing values and data consistency  

3. **Exploratory Analysis**
   - Distribution plots for features and quality  

4. **Correlation Analysis**
   - Correlation matrix and heatmap visualization  

5. **Feature Insights**
   - Identified important variables affecting quality  

6. **Visualization**
   - Scatter plots, histograms, and heatmaps  

## Key Insights
- Alcohol content shows a positive relationship with wine quality  
- Volatile acidity negatively impacts quality  
- Most samples are clustered around mid-range quality scores
